# Feature: Setup Base e Layout Inicial

## Objetivo

Criar o esqueleto inicial do sistema com layout base reutilizavel e navegacao simples para evolucao das proximas telas.

## O que faz

Configura o `base.html` com Tailwind via CDN, HTMX via script global e uma estrutura visual com menu lateral (sidebar) para navegacao entre as funcionalidades.

## Comandos pra IA

- Configurar o template base com CDN do Tailwind CSS.
- Incluir script global do HTMX no layout base.
- Criar sidebar simples, com links navegaveis para telas atuais e futuras.
- Garantir semantica HTML com landmarks (`header`, `nav`, `main`, `footer`).
